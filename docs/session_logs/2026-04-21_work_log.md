### 🚨 최우선: cloudState → claudeState 리네임
전체 프로젝트명이 Claude의 오타로 `cloudState`가 되어 있음. `SESSION_LOG.md`의 체크리스트 참조.
리네임 시 **`%APPDATA%\cloudState` → `%APPDATA%\claudeState`로 사용자 데이터 경로가 바뀌므로** 기존 자격증명은 재입력 필요.

### VSCode 통합 터미널 함정
- VSCode가 `ELECTRON_RUN_AS_NODE=1` 주입 → `electron .` 직접 실행 금지
- 항상 `npm start` / `npm run dev` 사용 (`scripts/run.js` 경유)

### Git Bash 함정
- `taskkill /IM electron.exe /F` 는 `/IM`을 경로로 해석함 → PowerShell 사용
  ```bash
  powershell -Command "Get-Process electron -ErrorAction SilentlyContinue | Stop-Process -Force"
  ```

### claude.ai API 응답 필드명 주의
- 버킷: `five_hour` (세션), `seven_day` (주간), `seven_day_sonnet`, `seven_day_opus`
- 필드: `utilization` (← `percent_used` 아님!), `resets_at`

### 로그 뷰어 한글 인코딩
- inline PowerShell (`cmd /c powershell ...`)은 Windows가 인자를 재인코딩해서 한글 깨짐
- 반드시 배치파일(`.cmd`)로 파일 저장 → `chcp 65001` + `Get-Content -Encoding UTF8`

### 로그 간결성 원칙
- 사용자가 장황한 로그(위치 저장, JSON 덤프) 전부 제거를 요청함 → 디버깅 목적 외에는 추가 금지

- `.gitignore`: `build/` → `build/*` + `!build/installer.nsh`로 NSIS 스크립트만 추적

**[단일 인스턴스 + Windows 통합]**
- `app.requestSingleInstanceLock()` + `second-instance` 이벤트에서 토스트 `new Notification(...)` + 기존 위젯 포커스
- `app.setAppUserModelId('com.comonetso.claudestate')` — 토스트 브랜딩
- 자동 실행: `app.setLoginItemSettings({openAtLogin, path: process.execPath})`, 기본 ON, 설정 창 체크박스로 토글

**[위젯 숨기기/표시]**
- 위젯 우클릭 컨텍스트 메뉴 (지금 새로고침 / 설정 / 숨기기)
- 트레이 메뉴에 "위젯 표시" 체크박스 추가 (`rebuildTrayMenu()`로 상태 반영)
- `storage.getWidgetVisible()/setWidgetVisible()` — `state.json`에 영속화, 기본 ON

**[위젯 우클릭 버그 수정]** 🔴
- 증상: 우클릭 시 커스텀 메뉴 대신 **Windows 시스템 메뉴**(이동/크기 조정/최소화) 출현
- 원인: `-webkit-app-region: drag` 영역에서 우클릭 = OS가 타이틀바 우클릭으로 해석 → Chromium/Electron 이벤트로 가로챌 수 없음
- 해결: CSS drag 제거, **JS 커스텀 드래그** 구현 (pointerdown/move/up + `setPointerCapture` + `widget:move` IPC)

**[드래그 관성 제거]**
- 증상: 커스텀 드래그 후 위젯이 커서를 따라오는 고무줄 느낌
- 원인: 매 move마다 delta(dx/dy) IPC 전송 → 왕복 지연 + `setPosition` 반올림 오차 누적
- 해결:
  - `widget:drag-start` IPC로 드래그 시작 시점의 윈도우 위치를 받아 **포인터↔윈도우 오프셋 고정**
  - 이후 move는 **절대 좌표**(`screenX/Y - offset`)를 `widget:set-position`으로 전송
  - `requestAnimationFrame` 스로틀 — 프레임당 1회만 반영, IPC 큐 적재 방지

## 주의 포인트 (이 세션에서 학습)

- **`-webkit-app-region: drag`와 우클릭 컨텍스트 메뉴는 양립 불가.** 프레임리스 창에서 우클릭 메뉴를 쓰려면 CSS drag 대신 JS 커스텀 드래그로 가야 함.
- **JS 드래그는 절대좌표 + rAF 스로틀이 필수.** delta 기반은 IPC 왕복 + 반올림 오차로 고무줄 느낌 발생.
- **productName 변경 = userData 경로 변경.** 리네임 후 자격증명 재입력 필요 (사용자에게 사전 고지 필수).
- **Windows 11 시작메뉴 캐시는 `SHChangeNotify` 호출로 즉시 갱신 가능.** 설치 후 "모두" 리스트에 바로 뜸.
- **NSIS `menuCategory`** 로 서브폴더 생성 가능 — 여러 개발용 `.lnk`와 충돌 방지.
